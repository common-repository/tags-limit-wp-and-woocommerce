Limit the nummber of Tags in Default Widget and WooCommerce Tags Widget. 

== Description ==

Limit the nummber of Tags in Default Widget and WooCommerce Tags Widget. You can easily add the number of tags cloud to display on the default widget of WordPress and WooCommerce. You can easily manage those tags from Customizer settings. Go to Appearance > Customize > Tag Options to check the available options. 

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'tags-limit-wp-and-woocommerce'
3. Activate Tags Limit WP and WooCommerce plugin from your Plugins page. 
4. Go to the Customizer (Appearance -> Customize-> Tags Limit WP and WooCommerce) to change the number of tags in widget to your site.

= From WordPress.org =

1. Download tags-limit-wp-and-woocommerce.
2. Upload the 'tags-limit-wp-and-woocommerce' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate tags-limit-wp-and-woocommerce from your Plugins page. 
4. Go to the Customizer (Appearance -> Customize) to change the number of tags in widget to your site.
